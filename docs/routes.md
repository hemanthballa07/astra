# Astra Routes

## V1 Routes
- /                 Home
- /profiles         Profile selection
- /anime            Anime hub
- /series           Series hub
- /movies           Movies hub
- /browse           Category browse / filters
- /title/[slug]     Title details
- /watch/[slug]     Player
- /my-list          My List
- /search           Search

## Route Responsibilities

### /
- hero spotlight
- continue watching
- trending rails
- anime rail
- movie rail
- upcoming highlights
- genre chips

### /profiles
- choose profile
- add profile
- edit profile entry point

### /anime
- seasonal anime
- top airing
- latest episodes
- genre / studio / theme filters
- sub / dub toggle

### /series
- trending series
- continue watching series
- genre / year filters

### /movies
- trending movies
- popular movies
- genre / year filters

### /browse
- filter sidebar
- sort controls
- mixed content grid

### /title/[slug]
- hero art
- poster
- metadata
- watch CTA
- season selector
- episode list
- related titles

### /watch/[slug]
- player
- subtitle / audio controls
- next up
- episode selector
- progress save state

### /my-list
- saved titles grid
- tabs for anime / series / movies
- sort options

### /search
- query input
- filters
- results grid

## Not in V1
- /community
- /watch-together
- /random
