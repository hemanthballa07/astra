# Astra Design System

## Brand Direction
Netflix-inspired cinematic streaming UI for anime, series, and movies.

## Principles
- content first
- dark immersive surfaces
- strong hierarchy
- reusable components
- responsive by default

## Typography
- Heading font: Cabinet Grotesk
- Body/UI font: Outfit

## Color Tokens
- bg.primary: #050811
- bg.card: #0d121f
- accent.primary: #8b5cf6
- accent.secondary: #d946ef
- text.primary: #ffffff
- text.secondary: rgba(255,255,255,0.72)
- border.subtle: rgba(255,255,255,0.08)

## Radius
- sm: 8px
- md: 12px
- lg: 16px
- xl: 24px

## Component Rules
- one shared top navigation
- portrait cards for titles
- landscape cards only for continue watching / progress
- primary, secondary, and ghost button variants
- compact metadata chips
- progress bars only where playback context exists

## Interaction Rules
- subtle hover lift
- smooth opacity and transform transitions
- visible focus states
- no excessive glow or noisy motion
